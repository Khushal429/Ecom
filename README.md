# Shoe E-Commerce Website

## Project Structure
- front end/
- back end/

## Backend Setup
```bash
cd "back end"
npm install
npm run dev
```

## Frontend Setup
```bash
cd "front end"
npm install
npx live-server
```

## Deployment
Frontend:
- Netlify
- Vercel

Backend:
- Render
- Railway

## Important
Create a `.env` file in backend using `.env.example`.
